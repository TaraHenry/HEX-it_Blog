Highly Efficient X-it

A Real-Time Evacuation System

Overview

What is HEX-it?

HEX-it, is a mobile-device-mounted hardware-software system that captures real-time sensor data, to accurately map accessible and inaccessible areas within a building during emergencies, identify trapped individuals, and determine the safest evacuation routes.

How Does it Work?

  • The system will take in as input, real-time sensor data, a pre-loaded building blueprint, and real-time location data from emergency response personnel within the building.
  • Based on the sensor data, the hardware system will send actionable data to a control centre. This data includes environmental information, spatial mapping, and motion and obstacle detection.
  • The system will output to a control center, a website displaying a generated map of the building, the location and path to identified victims, and compromised areas noted on the map.

    • Why HEX-it?

    Current emergency response systems in the event of a natural disaster, are unaware of possible obstacles that could change the layout of the building, and the possible location of trapped individuals. This leads to delayed evacuations and inefficient rescue operations.

    There is a critical need for a system that can provide real-time situational awareness, including the location of trapped individuals, environmental conditions, and optimal evacuation routes, to reduce confusion and enhance the safety of persons and the effectiveness of emergency responders.

    Target Audience

    The system is designed for emergency responders and their teams, including fire departments, police, and other emergency services. It is also relevant to building management and safety personnel who are responsible for ensuring the safety of occupants during emergencies. The system can be used in various settings, where real-time situational awareness and efficient evacuation planning are critical.

    Features


    Hardware

    • Drone/Mobile-device mounted
    • Real-time sensor data capture such as:
      • Temperature and Altitude -- using BMP sensors
      • Motion and Orientation -- using MPU modules
      • Spatial Mapping -- using LiDAR
      • Image Capture for Human detection -- using ESP32 Camera
    • Real-time human detection and flagging.

    Software

    • Sensor Data Transmission and Processing
    • Evacuation Path Generation
    • Building Mapping - display and render of 2D spatial maps using incoming sensor data.
    • Real-time visualization data storage (mapped environments and detection logs)

    Communication

    • Wireless data transmission and local data processing using LoRa ESP module
    • Relay of Evacuation Instructions between Control Centre and Emergency Personnel

    User Interface

    • Web-based interface for real-time monitoring
    • Interactive map display of building layout
    • Alerts and notifications for emergency responders

    Web Interface

    Login Page

    Provides secure access to the interactive web-page for control centre personnel. It features two input fields, a username and password that are provided to the user upon acquisition of the system.

    Live Page

    Displays real-time data from the sensors and drones, providing comprehensive situational awareness for the control centre personnel.

    Why HEX-it?


    Real-time situational awareness

    Provides real-time information on the state of the environment, allowing for more informed decision-making by emergency responders. This in turn, enhances the overall safety and efficiency of rescue operations.

    Enhanced Evacuation Efficiency

    Identifies the shortest and relatively safest evacuation routes based on the condition of the environment, reducing evacuation time and confusion.

    Adaptability

    Adapts to changing environmental conditions like obstacle detection, ensuring optimal performance in dynamic situations.

    System Integration

    Integrates various subsystems and components into a cohesive system, ensuring seamless communication and data exchange.
